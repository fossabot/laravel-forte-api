## Git Rules
### Issue Information 
- 이슈 라벨
    - hotfix: 버그 발생 시 이슈 생성 후 해당 라벨을 추가합니다.
    - database: 데이터베이스 관련된 추가/오류는 해당 라벨을 추가합니다.
    - develop: 개발을 포함한 이슈에는 해당 라벨을 추가합니다.
    - feature: 신규 개발 시 해당 라벨을 추가합니다.
